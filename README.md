# Mapper Map Recipe Patch

This is an (incredibly) lightweight patch that simply changes the recipe
of the map from Genhis's mod Mapper to allow the compass to be "returned"
when a map is made.

## Features

* **A patch** to modify the **Map** recipe.

## Dependencies

Mapper (
    https://github.com/Genhis/VintageStoryMods; Mapper
    https://mods.vintagestory.at/mapper
)
